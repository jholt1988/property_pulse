# Keyring OS Starter

A production-minded **Next.js + Tailwind** starter for **Keyring OS — The Operating System for Real Estate**.

## Included

- App Router structure
- Dark-mode control-plane dashboard shell
- Shared design tokens
- Core UI primitives
- Sidebar + top bar
- Metric cards
- Property data table
- AI insight cards
- Approval queue rows
- Starter module routes:
  - Dashboard
  - Inspect
  - Lease
  - Finance
  - AI
  - Tenants
  - Properties

## Run locally

```bash
npm install
npm run dev
```
