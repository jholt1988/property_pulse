## Weekly Board

## Week 1 — Foundation + P0 Sprint Kickoff
**Primary Goal:** lock governance, claims, and execution ownership.

- [ ] Establish claims register (Verified/Estimate/Future/Unverified) — **Owner: PMM**
- [ ] Freeze canonical marketing source doc and deprecate duplicate — **Owner: PMM + CL**
- [ ] Patch all placeholders (links, contacts, CTA URLs) in launch assets — **Owner: CL + LM**
- [ ] Kick off P0 security remediation tracker — **Owner: SEC**
- [ ] Kick off P0 ops closure tracker (prod config/backup/rollback) — **Owner: OPS**
- [ ] Kick off P0 QA smoke tracker and dependency list — **Owner: QA**
- [ ] Finalize channel owner RACI and publish cadence SLA — **Owner: PMM**

**Exit Criteria:** claims register live; owners assigned; P0 trackers active.

---

## Week 2 — Proof + Asset Production
**Primary Goal:** make proof and media launch-ready.

- [ ] Build claim→evidence appendix v1 — **Owner: PMM + AN**
- [ ] Assemble media kit v1 (hero image, social cards, screenshot set) — **Owner: DL**
- [ ] Draft website launch page with launch-safe copy — **Owner: GMS + CL**
- [ ] Build email sequence v1 (announce, proof, CTA) — **Owner: LM**
- [ ] Draft webinar #1 deck and demo flow — **Owner: PMM + SE**
- [ ] Run P0 checkpoint review (SEC/OPS/QA) — **Owner: PMM**

**Exit Criteria:** proof appendix + media kit v1 ready; launch page draft complete.

## Week 3 — Prelaunch Demand Warming (No GA claims)
**Primary Goal:** start audience warming with safe educational content.

- [ ] LinkedIn cadence starts (3 posts) — **Owner: SL + CL**

- [ ] Publish SEO article #1 and #2 — **Owner: GMS**
- [ ] Send nurture email #1 — **Owner: LM**
- [ ] Launch webinar registration page (internal/soft audiences) — **Owner: GMS + PMM**
- [ ] Sales enablement session #1 (messaging + objections) — **Owner: SEL**

**Exit Criteria:** consistent prelaunch publishing and first engagement baseline captured.

---

## Week 4 — Gate Review + Launch Readiness Freeze
**Primary Goal:** confirm GO/NO-GO for external launch phase.

- [ ] Validate P0 closures with evidence links — **Owner: SEC + OPS + QA**
- [ ] Final launch content pack QA (copy/legal/compliance) — **Owner: PMM**
- [ ] Validate pricing/contact endpoints — **Owner: PMM + OPS**
- [ ] Final webinar rehearsal and recording backup — **Owner: PMM + SE**

**Exit Criteria:** GO approval required to begin Week 5 external launch motions.

## Gate Rules (must remain visible in board)
- No full launch activity before P0 security + ops + QA closure evidence.
- No quantified public claim without verification tag and evidence link.
- No pricing/contact-led CTA push before endpoint validation.
